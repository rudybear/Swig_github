Сборка и запуск под CSHARP

1. Необходимо скачать SWIG (http://www.swig.org/). Прописать путь к папке где лежит swig.exe в PATH переменную. Либо поменять ciustom build step, чтобы exe вызывался из нужного места
2. В папку bin\Debug переписать движок 
3. Загрузить солюшен и собрать.


Пример минимальный - ничгео особо полезного не делает, но грузит Core2 и создает ICore интерфейс и демонстриурет принцип работы в целом.
Настойчиво рекомендую посмотреть tutorial на сайте SWIG,  который объясняет принципы работы.
